  1. Cars Simulator

    2. This Java program is basically about using Apache kafka messaging system to send messages to another Java program (see Cars Dashboard) in real time. At the beginning all trips are loaded from a csv file in disk into lists in memory, to decrease i/o operation overhead. When a button is clicked, a thread is created to send the corresponding trip information through kafka producer. Java swing is used to draw the graphical user interface GUI.

  2. Cars Dashboard

    3. This is another Java program that utilizes Apache kafka consumer to receive messages from Cars Simulator. It is meant to receive information about each trip and then display it onto real map. Google maps are used to locate cars position on it. There is a legend located on the right hand side to display helpful information about each car trip such as speed, rpm and fuel consumption. Javafx is used to draw the graphical user interface GUI.
